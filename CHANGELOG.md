# Changelog

Все значимые изменения этого проекта будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/) и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [0.2.0] - 2025-05-03
### Добавлено
- Добавлен новый класс `TaskManager` - `FileBackedTaskManager`
- Добавлен утилитарный класс по работе с CSV строками. Реализована частичная поддержка спецификации CSV (RFC 4180) - не поддерживаются "многострочные" поля.
- Добавлена возможность сохранение данных в формате CSV для `FileBackedTaskManager`
- Добавлена возможность восстановление данных из формата CSV для `FileBackedTaskManager`
- Добавлена сериализация `Task`, `Epic` и `SubTask` согласно CSV формату.
- Добавлена обработка исключений при работе с внешними файлами.
- Добавлены тесты для проверки работы обновления - `FileBackedTaskManagerTest`.
- Добавлены тесты для проверки корректности парсинга CSV строк с учетом исключений.

### Изменено
- Автоматическое сохранение данных в файле при работе с задачами типа `Task`, `Epic` и `SubTask` при операциях создания, изменения и удаления.
- Улучшен метод `updateStatus` класса `TaskManager`
- Добавлены дополнительные тесты для проверки корректности работы `updateStatus`
- Различные мелкие исправления

### Удалено
- —

## [0.1.0] - 2025-04-21

- Базовая структура проекта
- Классы: `Task`, `Epic`, `SubTask`
- Интерфейсы: `TaskManager`, `HistoryManager`
- Реализация хранения задач в памяти (`InMemoryTaskManager`)
- Логика автоматического обновления статуса эпика
- История просмотров задач (`InMemoryHistoryManager`)
