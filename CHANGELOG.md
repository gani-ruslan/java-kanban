# Changelog

Все значимые изменения этого проекта будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/) и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [0.1.0] - 2025-04-21
### Добавлено
- Базовая структура проекта
- Классы: `Task`, `Epic`, `SubTask`
- Интерфейсы: `TaskManager`, `HistoryManager`
- Реализация хранения задач в памяти (`InMemoryTaskManager`)
- Логика автоматического обновления статуса эпика
- История просмотров задач (`InMemoryHistoryManager`)
- Класс `Main` как точка входа

### Изменено
- Улучшена структура пакетов и распределение ответственности между классами
